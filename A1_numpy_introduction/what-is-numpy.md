# what is NUMPY?

 Numpy is a fundamental package for scientific calculation in python. It Provides a multi-dimensional array objects and various derived objects (like masked arrays and matrices).
 It Enables us to do some simple mathematical operations and logical , shape maupulations also. for further we can do so much like basic linear algebra, stastical operations , and generating random simulations.
